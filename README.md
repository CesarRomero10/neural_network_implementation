# neural_network_implementation

Descripción:
Este repositorio se desarrolla como parte de las actividades de evaluación del Diplomado en Ciencia de Datos impartido por la Universidad Nacional Autónoma de México. El objetivo central es demostrar la aplicación de redes neuronales artificiales utilizando la biblioteca TensorFlow para resolver problemas de aprendizaje profundo, abarcando tanto la regresión como la clasificación de datos.
